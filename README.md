# luma-adsb

Designed to run on [ADSB.im](https://adsb.im) and [GeeekPi](https://mygeeekpi.com/) [Mini Tower Kit for Raspberry Pi 5](https://www.amazon.com/dp/B0CQYTN94R)

Displays the ADSB data on the SSD1306 display
